# GLOSSARY: agentRoadmap.md

## Core Concepts
- **State:** A defined point in the project lifecycle. States are nodes in a Directed Acyclic Graph (DAG).
- **Terminal State:** A high-level goal or permanent destination node.
- **Transitional State:** A significant milestone reached on the path to a goal.
- **Operational State:** A repeatable state or machine-driven step in the DAG.
- **Scouting:** The process of an agent role analyzing the current project state and mission to identify future states and obstacles.
- **Mapping:** The act of adding discovered states and dependencies to the project map (`MAP.md` and `nodes/`).
- **Reaching:** The act of an agent role performing the technical work required to satisfy a state's "Proof of Arrival."
- **Proof of Arrival:** Terminal output or technical validation demonstrating a state has been reached.
- **Hype:** A concise, impact-oriented promotional statement for a reached state.
